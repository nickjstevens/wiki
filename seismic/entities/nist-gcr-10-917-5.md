# NIST GCR 10-917-5

NIST GCR 10-917-5 is a NEHRP technical brief on nonlinear structural analysis for seismic design. It is a practitioner-facing bridge between theory, modeling assumptions, and the level of interpretation needed when nonlinear analysis results are used for design decisions.

## Why it matters

- It gives a practical engineering frame for when nonlinear analysis adds value and when it can be misused.
- It supports this wiki's distinction between analysis capability and dependable design judgment.
- It is a key reference for [[nonlinear-seismic-assessment-methods]] and [[pushover-analysis-for-seismic-assessment]].
